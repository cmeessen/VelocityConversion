Installation¶

Prerequisites¶

VelocityConversion requires Python 2.7 or 3, and a few modules. I recommend to use Python 3 and Pipenv.

With pipenv¶

The advantage of pipenv is, that all dependencies and their versions are specified in the Pipfile.lock. Pipenv will create an environment where all necessary packages will be installed. To install dependencies with pipenv, navigate to the project root folder, and run

pipenv install

To install other dependencies required to compile the documentation, run:

pipenv install --dev

Enter the environment in the terminal by running

pipenv shell

With pip¶

or if you are using a Python version other than Anaconda:

pip install numpy

and to be able to compile the documentation:

pip install sphinx m2r sphinx_rtd_theme

With Anaconda¶

Install the required modules with Anaconda:

conda install -c conda-forge numpy

To be able to build the documentation from source, further packages are required:

conda install -c conda-forge ipython sphinx m2r sphinx_rtd_theme

Installation¶

VelocityConversion can be used both as a command line tool and as a Python module. In both cases, the first step is to create a clone or download the repository.

git clone https://github.com/cmeessen/VelocityConversion.git

and install it with

pip install .

You can also use VelocityConversion without installing it. In this case, you need to add the path of the ./VelocityConversion folder to the system path when importing the module:

import sys
sys.path.append('../path/to/gitrepo')

Run tests¶

To check whether everything is running correctly, enter the newly created folder and run:

python test.py

If the output looks like this, everything is working well:

test_vp_AlphaConst (__main__.TestVelocityConversion) ... ok
test_vs_AlphaConst (__main__.TestVelocityConversion) ... ok
test_vs_AlphaPT (__main__.TestVelocityConversion) ... ok
test_vs_AlphaT (__main__.TestVelocityConversion) ... ok

----------------------------------------------------------------------
Ran 4 tests in 1.633s

OK

Usage as a Python module¶

VelocityConversion can be used as a Python module. Therefore, navigate to the folder that contains your clone of the repository (and [setup.py](./setup.py)) and execute

pip install -e .

Now, the module can be imported to Python:

from VelocityConversion import MantleConversion
MC = MantleConversion()

Detailed instructions on the module usage a given in Python module.

Usage as a command line tool¶

VelocityConversion can also be run from the terminal. Please refer to Terminal application for more information. Using it as a terminal application does not require a pip installation.
